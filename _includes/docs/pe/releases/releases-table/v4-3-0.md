### ThingsBoard PE v4.3.0 (Jan 20, 2026)

Everything from [TB CE v4.3](https://github.com/thingsboard/thingsboard/releases/tag/v4.3) with the following improvements and bug fixes.

**Major Improvements**

  * New licensing model and add-ons support;

**Minor Improvements**

  * Added support for custom icon in custom menu;
  * Added new "Schedule" column on scheduler events page;
  * Added endpoint field for OPC-UA integration;
  * Edge Report templates and scheduler events sync;
  * Line chart report component improvements;
  * New 'user groups' assignee type for custom menu;
  * Prohibited current user's group deletion;

**Bug fixes**

  * Fixed outdated owners cache after assigning device to another tenant;
  * Fixed table rendering with column "values" in reports;
  * Fixed reports and report templates not displayed with EDQS;
  * Fixed missing extension in report file name;
  * Fixed column order for "On cell click" action for Entities Table widget;
  * Fixed Timeseries Table export to use entity label instead of name;
  * Fixed Line break displaying in the Advanced Feature Schedule column;
  * Fixed timestamp column formatting for XLSX file export in timeseries table;
  * Fixed columns order on export table widgets data;
  * Fixed error after trying to save report details after first try;
  * Fixed color adjustment for Activation Link Expired page;
  * Fixed integration converter library autocompletes;
  * Fixed installation and upgrade configuration of the Edge PE;
