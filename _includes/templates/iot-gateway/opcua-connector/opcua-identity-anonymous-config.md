This option of identity subsection is the simplest.

| **Parameter** | **Default value** | **Description**                    |
|:--------------|:------------------|------------------------------------|
| type          | **anonymous**     | Type of identity on OPC-UA server. |
| ---           |                   |                                    |

This part of configuration will look like this:  

```json
    "identity": {
      "type": "anonymous"
    },
```
