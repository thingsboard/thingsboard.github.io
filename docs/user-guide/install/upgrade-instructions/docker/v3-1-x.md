### Upgrading ThingsBoard CE to 3.1.1

**NOTE**: These upgrade steps are applicable for ThingsBoard version 3.1. In order to upgrade to 3.1.1 you need to [**upgrade to 3.1CE first**](#upgrading-thingsboard-ce-to-31).

#### ThingsBoard image download

Download ThingsBoard Docker image:

```bash
docker pull thingsboard/tb-postgres:3.1.1
```
{: .copy-code}

#### ThingsBoard service upgrade

* Stop ThingsBoard `mytb` container if it is running.

```bash
docker compose stop mytb
```
 {: .copy-code}

{% capture dockerComposeStandalone %}
If you still rely on Docker Compose as docker-compose (with a hyphen) execute next command:
<br>**docker-compose stop mytb**
{% endcapture %}
{% include templates/info-banner.md content=dockerComposeStandalone %}

* Update docker-compose.yml

```yml
< . . . >
  mytbce:
    image: "thingsboard/tb-postgres:3.1.1"
< . . . >
```
{: .copy-code}

* Change `upgradeversion` variable:

```bash
echo '3.1.0' | sudo tee ~/.mytb-data/.upgradeversion
```
{: .copy-code}

* Execute the following commands to upgrade ThingsBoard to the new version:

```bash
docker run -it -v ~/.mytb-data:/data --rm thingsboard/tb-postgres:3.1.1 upgrade-tb.sh
```
{: .copy-code}

* Start ThingsBoard:

```bash
docker compose up -d
```
{: .copy-code}

{% capture dockerComposeStandalone %}

<br>**docker-compose up -d**
{% endcapture %}
