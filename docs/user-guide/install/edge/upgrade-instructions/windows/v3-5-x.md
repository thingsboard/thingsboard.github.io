## Upgrading Edge CE to 3.5.1.1

{% assign updateServerLink = "windows/v3-5-x/#upgrading-thingsboard-ce-to-3511" %}
{% assign previousVersion = "3.5.1" %}
{% include templates/edge/install/compatibility-warning-version.md %}

### Download the Edge package

Download ThingsBoard Edge package for Windows: [tb-edge-windows-3.5.1.1 zip](https://github.com/thingsboard/thingsboard-edge/releases/download/v3.3.5.1.1/tb-edge-windows-3.5.1.1.zip).

Stop the Edge service:

```bash
net stop tb-edge
```
{: .copy-code}

Make a backup of your ThingsBoard Edge configuration located in **\<ThingsBoard Edge install dir\>\conf** (for example, **C:\tb-edge\conf**).

Extract and update configuration:
* Extract the ThingsBoard Edge package.
* Compare your old configuration files (from the backup) with the new ones.
* Merge any custom settings into the new configuration files.

### Run the upgrade script

**Note:** The upgrade script must be run with Administrator privileges.

```bash
C:\tb-edge>upgrade.bat --fromVersion={{previousVersion}}
```
{: .copy-code}

### Start the service

```bash
net start tb-edge
```
{: .copy-code}
