## Upgrading Edge PE to 3.5.1.1

{% assign updateServerLink = "windows/v3-5-x/#upgrading-thingsboard-pe-to-3511" %}
{% assign previousVersion = "3.5.1" %}
{% assign peDocsPrefix = "pe/" %}
{% include templates/edge/install/compatibility-warning-version.md %}

### Download the Edge package

Download ThingsBoard Edge package for Windows: [tb-edge-windows-3.5.1.1pe.zip](https://dist.thingsboard.io/tb-edge-windows-3.5.1.1pe.zip).

### Stop the Edge service

```text
net stop tb-edge
```
{: .copy-code}

### Back up the configuration

Make a backup of your ThingsBoard Edge configuration located in **\<ThingsBoard Edge install dir\>\conf** (for example, **C:\tb-edge\conf**).

### Extract and update configuration

1. Extract the ThingsBoard Edge package.
2. Compare your old configuration files (from the backup) with the new ones.
3. Merge any custom settings into the new configuration files.

### Run the upgrade script

**Note:** The upgrade script must be run with Administrator privileges.

```text
C:\tb-edge>upgrade.bat --fromVersion={{previousVersion}}
```
{: .copy-code}

### Start the Edge service

```text
net start tb-edge
```
{: .copy-code}
