#### ThingsBoard PE service upgrade

1. Change the version of the `thingsboard/tb-pe-node` and `thingsboard/tb-web-report` in the `docker-compose.yml` file to the **3.9.0PE**.

2. Execute the following commands:

  ```bash
  docker pull thingsboard/tb-pe-node:3.9.0PE
  docker pull thingsboard/tb-web-report:3.9.0PE
  docker compose stop thingsboard-pe
  docker compose run --rm -e UPGRADE_TB=true -e FROM_VERSION="3.8.1" thingsboard-pe
  docker compose up -d
  ```
